## What This Task Does

### Assigns a theme to each review using your manual rules:

- Account Access

- Transaction Performance

- User Interface / UX

- Customer Support

- Feature Requests

Extracts all matched keywords from each rule and attaches them to the review record.

Generates enriched tables for:

Drivers per bank

Pain points per bank

Bank-level recommendations

Adds a new column keywords to all key reports so stakeholders understand what triggered each theme.
